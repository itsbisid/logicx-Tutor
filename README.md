# logicx-Tutor
Logic Reasoning Tutor is an interactive web app that simplifies learning discrete math through logic tools, dynamic quizzes, a maze game, and a concept-learning module. It generates truth tables, checks equivalence and validity, tracks performance, and supports accessible, self-paced learning.
